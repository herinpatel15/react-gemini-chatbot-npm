# react geminiAi chatbot

# Get start
`npm i react-gemini-chatbot-new`

## example code

```typescript
import { Chatbot } from 'react-gemini-chatbot-new'

function App() {
  return (
    <main style={{ height: "100vh" }}>
      <Chatbot apiKey={"--gemini api key--"} prompt={"hello gemini ai"} />
    </main>
  )
}

export default App
```
